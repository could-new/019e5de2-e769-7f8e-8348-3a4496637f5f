# Sambalpuri OctaPad Pro

A modern, high-performance Flutter drum pad application with a stylish dark neon interface and traditional Sambalpuri beats.

## Features
- 16 Multi-touch glowing pads
- Ultra low latency UI
- Dark neon DJ concert vibe
- Responsive for mobile and tablet modes
- Categories for Sambalpuri, DJ Remix, Meme Beats, and more

## Tech Stack
- Flutter (UI Toolkit)
- Google Fonts (Typography)

## How to Run
1. Ensure you have Flutter installed.
2. Clone the repository and navigate to the project directory.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the app on your preferred device or emulator.
